1. Login as Branch HR and complete salary activity from dashboard
  2. Logout and Login as admin
  3. Go to create payslip menu from sidebar
  4. Fill all the fields (month, year, out source, company branch), If you want to generate salary for some individuals then select for individuals radio button, else select for all and then press generate button
  5. A table will get loaded below the generate button. You can add/edit other addition and other deduction with remarks if required.
  6. Click Final Submit button (at the bottom of the page)
  7. Go to Payslip List menu, select month, branch and press submit button to see generated payslip, you can print, copy and download(in excel, csv and pdf format) the generated payslip
