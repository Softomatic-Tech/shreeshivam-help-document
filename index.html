1. Login as admin
  2. Go to create payslip menu from sidebar
  3. Fill all the fields (month, year, out source, company branch), If you want to generate salary for some individuals then select for individuals radio button, else select for all and then press generate button
  4. A table will get loaded below the generate button. You can add/edit other addition and other deduction with remarks if required.
  5. Click Final Submit button (at the bottom of the page)
